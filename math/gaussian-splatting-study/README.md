# 가우시안 스플래팅 학습 로드맵 (완전판)

## 학습 목표
중학교 1학년 수준의 수학 지식에서 시작하여 가우시안 스플래팅(Gaussian Splatting)을 이용한 3D 렌더링을 이해하는 데 필요한 **모든** 수학적 기초를 교과 과정처럼 단계적으로 학습합니다.

## 학습 철학
- **빠짐없이**: 필요한 모든 개념을 단계별로 학습
- **순차적으로**: 각 단계가 다음 단계의 기초가 됨
- **체계적으로**: 중학교 → 고등학교 → 대학교 수학 순서로
- **실용적으로**: 가우시안 스플래팅 이해에 필요한 내용 중심

## 학습 방법
- 각 주차는 독립적인 문서로 제공됩니다
- 이전 주차의 내용을 기반으로 새로운 개념을 배웁니다
- 각 문서는 `week-XXX.md` 형식으로 저장됩니다 (3자리)
- 예제, 그림, 연습문제를 통해 직관적으로 이해
- 각 Phase마다 복습 주차가 포함됩니다

## 전체 학습 계획 (60주 과정 = 약 1년 2개월)

> 💡 **학습 속도 조절**: 이해가 잘 안 되는 주차는 시간을 더 투자하세요. 천천히 확실하게!

## 💡 학습 팁

### 효과적인 학습을 위해
✅ **천천히**: 한 주차를 완전히 이해한 후 다음으로
✅ **손으로**: 예제를 종이에 직접 그려보고 계산하기
✅ **반복**: 이해 안 되면 여러 번 읽기
✅ **질문**: 막히면 언제든 질문하기
✅ **복습**: 정기적으로 이전 내용 되돌아보기
✅ **연결**: 새로운 개념이 이전 개념과 어떻게 연결되는지 생각하기

### 주의사항
❌ 이해 안 된 채로 넘어가지 않기
❌ 예제 건너뛰지 않기
❌ 연습 문제 안 풀고 넘어가지 않기
❌ 너무 빨리 진행하려 하지 않기
