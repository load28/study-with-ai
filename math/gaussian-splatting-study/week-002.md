## 1. 좌표계란 무엇인가?

좌표계는 **위치를 숫자로 표현하는 방법**입니다. 마치 보물 지도에서 "오른쪽으로 3칸, 위로 2칸"처럼 위치를 정확히 알려주는 방법이에요.

### 2D 좌표계의 구성

```
       y축 (세로)
       ↑
       |
   3   |
   2   |
   1   |
   0   |______________________→ x축 (가로)
       0   1   2   3   4   5
```

- **x축**: 가로 방향 (좌우)
- **y축**: 세로 방향 (상하)
- **원점**: 두 축이 만나는 점 (0, 0)

### 점의 표현

점은 **(x, y)** 형식으로 표현합니다.
- 첫 번째 숫자(x): 오른쪽으로 얼마나 갔는지
- 두 번째 숫자(y): 위로 얼마나 갔는지

**예제:**
- 점 A = (3, 2) → 오른쪽으로 3, 위로 2
- 점 B = (1, 4) → 오른쪽으로 1, 위로 4
- 점 C = (0, 0) → 원점

## 2. 음수 좌표

좌표는 음수도 될 수 있습니다!

```
       y축
       ↑
   2   |
   1   |
   0   |______→ x축
  -1   |
  -2   |
      -2  -1  0  1  2
```

- **음수 x**: 왼쪽으로 간 거리
- **음수 y**: 아래로 간 거리

**예제:**
- 점 D = (-2, 1) → 왼쪽으로 2, 위로 1
- 점 E = (1, -2) → 오른쪽으로 1, 아래로 2
- 점 F = (-3, -2) → 왼쪽으로 3, 아래로 2

### 4개의 사분면

```
   2사분면  |  1사분면
   (-x, +y) |  (+x, +y)
  __________|__________
   3사분면  |  4사분면
   (-x, -y) |  (+x, -y)
```

## 3. 거리 계산하기

두 점 사이의 거리는 어떻게 구할까요?

### 3.1 같은 선 위의 거리

**가로선 위의 두 점:**
점 A = (1, 2), 점 B = (5, 2)

```
  A -------- B
  (1,2)    (5,2)
```

거리 = |5 - 1| = 4

**세로선 위의 두 점:**
점 C = (3, 1), 점 D = (3, 5)

```
  D (3,5)
  |
  |
  |
  C (3,1)
```

거리 = |5 - 1| = 4

### 3.2 대각선 거리 (피타고라스 정리)

점 A = (1, 1), 점 B = (4, 5)를 생각해봅시다.

```
      B (4,5)
      |
      |
      |
      |_____
  A (1,1)
```

이것은 직각삼각형을 만듭니다!

1. **가로 거리** (x의 차이) = 4 - 1 = 3
2. **세로 거리** (y의 차이) = 5 - 1 = 4
3. **대각선 거리** = √(3² + 4²) = √(9 + 16) = √25 = 5

### 일반적인 거리 공식

두 점 P₁ = (x₁, y₁)과 P₂ = (x₂, y₂) 사이의 거리:

```
거리 = √[(x₂ - x₁)² + (y₂ - y₁)²]
```

**외울 필요 없어요!**
핵심은:
1. 가로 차이를 구한다
2. 세로 차이를 구한다
3. 각각 제곱해서 더한다
4. 루트를 씌운다

## 4. 실습 예제

### 예제 1: 점 찾기
다음 점들을 좌표로 표현하세요.

```
   5  |     B
   4  |
   3  |        C
   2  |
   1  |  A
   0  |____________
      0  1  2  3  4
```

**답:**
- A = (1, 1)
- B = (2, 5)
- C = (4, 3)

### 예제 2: 거리 계산
점 A = (2, 3)과 점 B = (6, 6) 사이의 거리는?

**풀이:**
1. 가로 차이: 6 - 2 = 4
2. 세로 차이: 6 - 3 = 3
3. 거리 = √(4² + 3²) = √(16 + 9) = √25 = 5

**답: 5**

### 예제 3: 정사각형 그리기
한 변의 길이가 3인 정사각형을 그려보세요.
왼쪽 아래 꼭짓점이 (1, 1)이라면 나머지 점들은?

**답:**
- 왼쪽 아래: (1, 1)
- 오른쪽 아래: (4, 1)
- 오른쪽 위: (4, 4)
- 왼쪽 위: (1, 4)

## 6. 실생활 응용

좌표는 어디에 사용될까요?

1. **게임**: 캐릭터의 위치, 적의 위치
2. **지도 앱**: 내 위치, 목적지 위치
3. **그래픽**: 화면에 그림 그리기
4. **3D 모델링**: 물체의 모든 점들의 위치
