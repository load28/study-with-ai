# 알고리즘 & 자료구조 학습 로드맵 (실전형)

## 학습 목표
파이썬 기본 문법부터 시작하여 **모든** 주요 알고리즘과 자료구조를 빠르게 마스터합니다. 코딩테스트와 실무에 필요한 문제 해결 능력을 체계적으로 기릅니다.

## 학습 철학
- **빠르고 실전적으로**: 이론만이 아닌 문제 풀이 중심
- **순차적으로**: 각 단계가 다음 단계의 기초가 됨
- **체계적으로**: 이론 → 구현 → 문제 풀이 순서로
- **실용적으로**: 코딩테스트와 실무에 바로 적용 가능

## 학습 방법
- 각 주차는 독립적인 문서로 제공됩니다
- 이론 설명 → 코드 예제 → 문제 풀이 순서
- 각 문서는 `week-XXX.md` 형식으로 저장됩니다 (3자리)
- 실제 코드 작성과 문제 풀이를 통해 실력 향상
- 각 Phase마다 실전 문제 풀이 주차 포함

## 전체 학습 계획 (38주 과정 = 약 9개월)

> 💡 **학습 속도 조절**: 이해가 잘 안 되는 주차는 시간을 더 투자하세요!

### 🚀 Phase 1: 파이썬 완전 정복 (1주차)
> 알고리즘 풀이에 필요한 모든 문법과 자료구조

**1주차: 파이썬으로 알고리즘 풀기**
- **기본 문법**
  - 변수, 자료형, 연산자
  - 조건문 (if, elif, else)
  - 반복문 (for, while, break, continue)
  - 함수 정의와 람다
  - 입출력 (input, print, sys.stdin)
  - 문자열 다루기
- **자료구조**
  - 리스트 (List) - 배열, 2D 배열
  - 튜플 (Tuple)
  - 딕셔너리 (Dictionary) - 해시맵
  - 집합 (Set)
  - 스택, 큐, 덱 (Deque)
  - 힙 (heapq)
  - 카운터 (Counter)
- **고급 기능**
  - 리스트 컴프리헨션
  - 정렬 (sort, sorted, key 함수)
  - 재귀 함수
  - itertools, collections 모듈
  - 비트 연산
- **실전 팁**
  - 입출력 최적화
  - 시간복잡도 기초
  - 자주 쓰는 패턴

### ⚡ Phase 2: 알고리즘 기초 (2-8주차)

**2주차: 시간복잡도와 공간복잡도**
- Big-O 표기법
- 시간복잡도 계산

- 알고리즘 효율성 분석
- 연습: 복잡도 분석 문제

**3주차: 정렬 알고리즘**

- 퀵 정렬 (Quick Sort)
- 힙 정렬 소개
- 파이썬 내장 정렬 활용
- 연습: 정렬 문제

**4주차: 이진 탐색 - 1부**
- 선형 탐색 vs 이진 탐색
- 이진 탐색 구현
- Lower Bound, Upper Bound
- 연습: 이진 탐색 기초 문제

**5주차: 이진 탐색 - 2부**
- 매개변수 탐색 (Parametric Search)
- 결정 문제로 변환하기
- 최솟값의 최댓값, 최댓값의 최솟값
- 연습: 이진 탐색 심화 문제

**6주차: 완전 탐색 (브루트 포스)**
- 완전 탐색의 개념
- 순열과 조합 (itertools)
- 비트마스크
- 연습: 완전 탐색 문제

**7주차: 투 포인터 & 슬라이딩 윈도우**
- 투 포인터 알고리즘
- 슬라이딩 윈도우
- 구간 합
- 연습: 투 포인터 문제

**8주차: Phase 2 실전 문제**
- 정렬, 탐색 종합 문제 15개
